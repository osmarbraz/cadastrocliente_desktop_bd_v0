Sistema de Cadastro de Clientes para Desktop

Este programa possui somente a classe Principal e a classe da interface gráfica. 

Toda iteração com banco de dados é tratada diretamente no tratamento de evento dos botões.

Crie o banco de dados antes de executar o projeto, as especificações das tabelas estão no arquivo banco.sql

A pasta src contêm os fontes do projeto e em lib o driver jdbc para MySQL.
